This project is a starter Next.js (app router) web application for a technology company focused on digital transformation.
It includes:

- Modern, responsive landing page layouts.
- 3D hero illustration via @react-three/fiber and @react-three/drei.
- Animated headlines and micro-interactions with framer-motion.
- Reusable components (Header, Footer, Hero, Services, IndustryGrid, CTA, SEO).
- Tailwind CSS for rapid responsive design.
- Basic SEO meta tags and structured content for indexability.
- Smooth scrolling, accessible forms, and mobile-first design.

Quick start
1. Install dependencies:
   npm install

2. Run dev server:
   npm run dev

3. Build production:
   npm run build
   npm start
